# 💿 OS Installation Guide (Windows)

## 🔍 Overview

Step-by-step guide to install Windows OS using bootable USB.

---

## 🧰 Requirements

* USB (8GB+)
* Windows ISO file
* Bootable tool (Rufus)

---

## ⚙️ Step 1: Create Bootable USB

1. Open Rufus
2. Select USB
3. Load Windows ISO
4. Click Start

---

## 🖥️ Step 2: BIOS Setup

1. Restart PC
2. Press `F2 / DEL / F12`
3. Set Boot Priority → USB

---

## 💻 Step 3: Install Windows

1. Select language
2. Click Install Now
3. Choose Custom Installation
4. Delete old partitions (careful ⚠️)
5. Select drive → Next

---

## 🔄 Step 4: Post Installation

1. Install drivers
2. Connect to internet
3. Run Windows Update
4. Install basic software

---

## 🧠 Tip

Always backup important data before installation.
